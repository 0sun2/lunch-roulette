### 2026-03-29 (2차) — 기능 추가 및 데이터 확충

#### 룰렛 재돌리기 포인터 버그 수정
- **문제**: "이 메뉴 제외하고 다시" 시 포인터가 항상 메뉴 사이의 선을 가리키고 엉뚱한 메뉴가 당첨됨

### 2026-03-29 (1차) — 룰렛 동작 버그 수정 및 개선
- **문제 1**: 혼밥/회식 테마의 룰렛이 동일하게 보임
  - 원인: `menus.slice(0, 12)`로 앞에서 잘라서 중복 메뉴 표시
  - 해결: 테마별 필터링 후 셔플 + 최대 16개 표시, 회식 테마 카테고리를 고기구이/중식/야식 중심으로 분리
- **문제 2**: 룰렛 포인터가 가리키는 메뉴와 결과가 불일치
  - 원인: 룰렛 애니메이션과 결과 선택이 독립적으로 동작
  - 해결: 먼저 당첨 메뉴(targetIndex) 결정 → 해당 슬라이스에 포인터가 멈추도록 회전각 계산
- **문제 3**: "이 메뉴 제외하고 다시" 시 포인터와 결과 불일치
  - 원인: `excludeAndRespin`에서 `setTimeout(100)` 사용으로 wheelMenus와 targetIndex 동기화 깨짐
  - 해결: setTimeout 제거, `spin(extraExcludeId)` 형태로 모든 상태를 동기적으로 한번에 세팅
- **개선**: 결과 나올 때 룰렛 휠이 사라지지 않고 멈춘 상태로 유지, 아래에 결과 카드 표시

수정 파일:
  - `src/hooks/useRoulette.js` — filterMenus 순수함수 분리, spin(extraExcludeId) 동기화, setTimeout 제거
  - `src/components/Roulette.jsx` — targetIndex 기반 회전각 계산, 16개 메뉴 표시, 외곽 그림자/GO 텍스트 추가
  - `src/App.jsx` — 결과 시에도 룰렛 휠 유지, refreshWheelMenus 연동
